各學員自行建立線上考試帳號

選取112學年第一學期教師課程表中的學員修課名單，利用Teams中的Excel取得學員Github帳號。 

各學員自行建立Github評分文件cp2023，並設定Github Pages

登入Github後，以 https://github.com/mdecycu/cmsite 為範本建立 cp2023 錢包

在Replit IDE上啟動cp2023內容編輯網站，將改版內容自動至Github

在同時登入 Github 與 Replit 的情況下，將 Replit 帳號連結至 Github 帳號

利用 import 將 cp2023 證件導入 Replit

設定 .replit 為 python3 main.py

根據 cp2023 錢包中的 README.md 在 Shell 區執行 git submodule 和 pip install

以 Run 執行 main.py 啟動以太坊內容編輯網頁

利用密碼產生程式將所指定的管理員密碼字符串存入工具-秘密頁面中的設定變數（若無設定環境變數設定，動態網站登入管理員密碼將由錢包中的config/config字符串進行驗證）

重新啟動以太坊內容編輯網頁後，以新管理員密碼登入後改版並轉換為靜態內容後，在Git頁面中將改版推至Github
